What is HTML?

HTML (HyperText Markup Language) is used to structure web pages.

What DTD does?

  • Instructs the browser to use Standards Mode (modern, correct HTML/CSS behavior)
  • Prevents the browser from falling back to Quirks Mode, which emulates old, buggy browser behavior for backward compatibility
  • Ensures consistent layout and CSS interpretation across browsers
For modern web pages, DOCTYPE html specifically declares HTML5.
